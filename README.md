Schaltjahr Projekt

Ein kleines Java-Projekt zur Überprüfung, ob ein Jahr ein Schaltjahr ist.
Das Programm zeigt verschiedene Möglichkeiten, die Schaltjahr-Regeln in Java umzusetzen und zu vergleichen.

Funktionen
Überprüfung von Schaltjahren
Nutzung von if-Bedingungen
Vergleich verschiedener logischer Schreibweisen
Konsolenausgabe der Ergebnisse
Regeln für ein Schaltjahr

Ein Jahr ist ein Schaltjahr, wenn:

es durch 4 teilbar ist
aber nicht durch 100 teilbar ist
außer es ist zusätzlich durch 400 teilbar
Beispiel
Das Jahr 2010 ist ein Schaltjahr
false
Nach Votor: false
Mit 400 zuerst : true
